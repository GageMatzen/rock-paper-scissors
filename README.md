
# Rock Paper Scissors

## Live Demo
[Play it here](https://gagematzen.github.io/rock-paper-scissors/)

A browser-based Rock Paper Scissors game built with vanilla JavaScript.

## How to Play

1. Click the Play it here link above.
2. Select your choice of Rock, Paper, or Scissors using the buttons shown.
3. Play until the first player reaches 5 points to see who wins!
